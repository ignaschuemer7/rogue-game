# Rogue Game 

A Python-based terminal dungeon crawler roguelike game developed as part of a programming course project.

## Quick Start

Navigate to the game directory and run:

```bash
cd templates/src/templates/oop/
python menu.py
```

## Project Documentation

For complete documentation, gameplay instructions, and technical details, see:
- [Full Game Documentation](templates/src/templates/oop/README.md)

## What is this game?

**Rogue Game** is a classic terminal-based dungeon crawler featuring:
- Multiple character classes (Barbarian, Knight, Ninja)
- Multi-level dungeons with procedural elements
- Combat system with various enemies
- Item collection and equipment system
- ASCII-based graphics
